# Electron Icons

Place your application icons here:

- `icon.png` - Linux icon (256x256 PNG)
- `icon.ico` - Windows icon (256x256 ICO)
- `icon.icns` - macOS icon (ICNS format)

For development, you can use online tools to convert PNG to ICO/ICNS formats.

Recommended icon sources:
- https://www.favicon-generator.org/
- https://icoconvert.com/
- https://cloudconvert.com/png-to-icns
